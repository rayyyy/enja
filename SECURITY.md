# セキュリティ

## サポートされるバージョン

現時点ではリリースブランチの区別がありません。**main ブランチ** 上の最新の状態を対象とします。重大な修正が入った場合は、このファイルまたはリリースノートで案内します。

## 脆弱性の報告

次のいずれかで、**非公開** にご連絡ください。

1. GitHub の **Security advisories**（リポジトリの「Security」タブから、可能であれば **Report a vulnerability**）
2. リポジトリ管理者（[@rayyyy](https://github.com/rayyyy)）へ GitHub 経由で連絡

## 公開 Issue に書かないでください

次のような情報は **Issue や Discussion に載せない** でください。

- Gemini **API キー** やトークン
- **クリップボード** に含まれうる個人情報・機密テキスト
- 再現手順に実データをそのまま含める必要がある場合は、ダミーに置き換えるか、報告経路を非公開にしてください

## スコープの目安

- アプリが意図せず **ローカル以外** に秘密を送信する、などの問題
- **依存パッケージ** に起因する、利用者に影響しうる脆弱性

Google Gemini API 側の障害や利用規約は、Google のサポート・ドキュメントを参照してください。
